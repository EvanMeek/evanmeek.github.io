添加必要源-視頻解碼器

  • packman源

packman源即第三方源，其包含了大量的軟件，類似ArchLinux的ArchlinuxCN源。 https://mirrors.tuna.tsinghua.edu.cn/packman/suse/openSUSE_Tumbleweed/

  • 多媒體解碼器

由於版權問題，openSUSE默認不自帶視頻解碼器，所以必須安裝以下包才能讓視頻等內容正常使用。 

zypper install --allow-vendor-change ffmpeg lame gstreamer-plugins-bad gstreamer-plugins-ugly gstreamer-plugins-ugly-orig-addon gstreamer-plugins-libav libavdevice56 libavdevice58 libdvdcss2 vlc-codecs

一些有用的包

obs源下載工具

zypper install opi

fcitx5

# 添加M17N倉庫，此命令的M17N代表這個倉庫的別名
zypper addrepo https://download.opensuse.org/repositories/M17N/openSUSE_Tumbleweed/ M17N
# 卸載fcitx相關包，*代表貪婪匹配，意味着所有fcitx前綴包名的包都會被刪除。
zypper remove "fcitx*"
# 安裝fcitx5
zypper install fcitx5 fcitx5-chinese-addons fcitx5-chinese-addons-devel fcitx5-configtool fcitx5-gtk2 fcitx5-qt-devel fcitx5-qt5 fcitx5-rime
